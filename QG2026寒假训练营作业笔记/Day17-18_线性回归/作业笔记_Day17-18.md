## 一、线性回归基础

### 1.1 模型假设
$$\hat{y} = \vec{w}^T \vec{x} + b$$

矩阵形式（增广）：
$$\hat{y} = X\vec{\theta}, \quad X \in \mathbb{R}^{N \times (d+1)}, \quad \vec{\theta} = (b, w_1, w_2, \ldots, w_d)^T$$

### 1.2 损失函数

**均方误差（MSE）+ L2 正则化：**
$$\mathcal{J}(\vec{\theta}) = \underbrace{\frac{1}{2N}\sum_{i=1}^{N}(\hat{y}_i - y_i)^2}_{\text{MSE}} + \underbrace{\frac{\lambda}{2N}\|\vec{w}\|^2}_{\text{L2 正则}}$$

> **为什么乘 1/2？** 求导后消掉系数 2，公式更简洁。

---

## 二、解析解（最小二乘法）

### 2.1 推导
对损失函数关于 $\vec{\theta}$ 求导并令其等于零：

$$\nabla_\theta \mathcal{J} = 0 \Rightarrow \vec{\theta}^* = (X^TX + \lambda I)^{-1} X^T \vec{y}$$

**无正则化版本：** $\vec{\theta}^* = (X^TX)^{-1} X^T \vec{y}$

> **注意：** $X^TX$ 可能接近奇异（列共线时），加 $\lambda I$ 可以保证可逆，同时等价于 L2 正则。

### 2.2 代码实现
```python
def analytical_solution(X, y, lambda_reg=0.0):
    """
    最小二乘解析解
    X: (N, d) 特征矩阵（不含偏置列）
    y: (N,) 标签
    lambda_reg: L2 正则化系数
    """
    N, d = X.shape
    # 增加偏置列（全1列）
    X_b = np.column_stack([np.ones(N), X])  # (N, d+1)
    
    # 正则化矩阵（不对偏置项正则化）
    reg_matrix = lambda_reg * np.eye(d + 1)
    reg_matrix[0, 0] = 0  # 不惩罚偏置
    
    # 解析解
    theta = np.linalg.solve(X_b.T @ X_b + reg_matrix, X_b.T @ y)
    
    b = theta[0]
    w = theta[1:]
    return w, b
```

## 三、梯度下降求解

### 3.1 梯度推导

$$\frac{\partial \mathcal{J}}{\partial \vec{w}} = \frac{1}{N}X^T(X\vec{w} + b\vec{1} - \vec{y}) + \frac{\lambda}{N}\vec{w}$$
$$\frac{\partial \mathcal{J}}{\partial b} = \frac{1}{N}\sum_{i=1}^{N}(\hat{y}_i - y_i)$$

### 3.2 完整实现（sklearn 风格）
```python
"""Day 17-18 练习：线性回归实战  日期：2026年3月2日-3日
要求：仅用 numpy，实现解析解（最小二乘法）和数值解（梯度下降）
sklearn 风格接口: fit/predict/score  评估: MSE/MAE/R²"""

import numpy as np
import matplotlib.pyplot as plt

def train_test_split(X, y, test_size=0.2, seed=42):
    """手动实现数据集划分"""
    np.random.seed(seed)
    N = len(y)
    indices = np.random.permutation(N)
    split = int(N * (1 - test_size))
    train_idx, test_idx = indices[:split], indices[split:]
    return X[train_idx], X[test_idx], y[train_idx], y[test_idx]

def normalize(X_train, X_test):
    """Z-score 标准化（仅用训练集统计量）"""
    mu = X_train.mean(axis=0)
    sigma = X_train.std(axis=0) + 1e-8  # 防止除零
    return (X_train - mu) / sigma, (X_test - mu) / sigma


class LinearRegression:
    """多元线性回归（numpy 实现）"""
    
    def __init__(self, lr: float = 0.01, n_epochs: int = 1000,
                 lambda_reg: float = 0.01, method: str = "gradient_descent"):
        """
        lr: 学习率
        n_epochs: 迭代次数
        lambda_reg: L2 正则化系数
        method: "gradient_descent" 或 "analytical"
        """
        self.lr = lr
        self.n_epochs = n_epochs
        self.lambda_reg = lambda_reg
        self.method = method
        self.w = None
        self.b = None
        self.loss_history = []
    
    def fit(self, X: np.ndarray, y: np.ndarray):
        """训练模型"""
        N, d = X.shape
        
        if self.method == "analytical":
            self.w, self.b = self._analytical(X, y)
        else:
            self._gradient_descent(X, y)
        return self
    
    def _analytical(self, X, y):
        """解析解"""
        N, d = X.shape
        X_b = np.column_stack([np.ones(N), X])
        reg = self.lambda_reg * np.eye(d + 1)
        reg[0, 0] = 0
        theta = np.linalg.solve(X_b.T @ X_b + reg, X_b.T @ y)
        return theta[1:], theta[0]
    
    def _gradient_descent(self, X, y):
        """梯度下降"""
        N, d = X.shape
        # 随机小初始化
        self.w = np.zeros(d)
        self.b = 0.0
        
        for epoch in range(self.n_epochs):
            y_pred = X @ self.w + self.b
            error = y_pred - y
            
            # 计算梯度
            dw = (X.T @ error) / N + (self.lambda_reg / N) * self.w
            db = error.mean()
            
            # 更新参数
            self.w -= self.lr * dw
            self.b -= self.lr * db
            
            # 记录损失
            mse = (error**2).mean() / 2
            l2 = (self.lambda_reg / (2 * N)) * np.sum(self.w**2)
            self.loss_history.append(mse + l2)
    
    def predict(self, X: np.ndarray) -> np.ndarray:
        return X @ self.w + self.b
    
    def score(self, X, y):
        """返回 R² 分数"""
        y_pred = self.predict(X)
        ss_res = ((y - y_pred)**2).sum()
        ss_tot = ((y - y.mean())**2).sum()
        return 1 - ss_res / ss_tot


def evaluate(y_true, y_pred, label=""):
    """计算 MSE、MAE、R²"""
    mse = np.mean((y_true - y_pred)**2)
    mae = np.mean(np.abs(y_true - y_pred))
    ss_res = ((y_true - y_pred)**2).sum()
    ss_tot = ((y_true - y_true.mean())**2).sum()
    r2 = 1 - ss_res / ss_tot
    
    print(f"\n{label}")
    print(f"  MSE  = {mse:.4f}")
    print(f"  MAE  = {mae:.4f}")
    print(f"  R²   = {r2:.4f}")
    return mse, mae, r2


# ── 测试：加州房价数据集 ──
if __name__ == "__main__":
    from sklearn.datasets import fetch_california_housing
    
    data = fetch_california_housing()
    X, y = data.data, data.target
    
    X_train, X_test, y_train, y_test = train_test_split(X, y)
    X_train_n, X_test_n = normalize(X_train, X_test)
    
    # 方法1：梯度下降
    gd_model = LinearRegression(lr=0.1, n_epochs=500, lambda_reg=0.01,
                                 method="gradient_descent")
    gd_model.fit(X_train_n, y_train)
    y_pred_gd = gd_model.predict(X_test_n)
    evaluate(y_test, y_pred_gd, "梯度下降法")
    
    # 方法2：解析解
    an_model = LinearRegression(lambda_reg=0.01, method="analytical")
    an_model.fit(X_train_n, y_train)
    y_pred_an = an_model.predict(X_test_n)
    evaluate(y_test, y_pred_an, "解析解（最小二乘）")
    
    # 对比 sklearn
    from sklearn.linear_model import Ridge
    sk_model = Ridge(alpha=0.01)
    sk_model.fit(X_train_n, y_train)
    y_pred_sk = sk_model.predict(X_test_n)
    evaluate(y_test, y_pred_sk, "sklearn Ridge")
    
    # 可视化损失曲线
    plt.figure(figsize=(8, 4))
    plt.plot(gd_model.loss_history, color="#E74C3C", linewidth=2)
    plt.title("梯度下降损失曲线")
    plt.xlabel("Epoch")
    plt.ylabel("Loss")
    plt.grid(True, alpha=0.3)
    plt.tight_layout()
    plt.savefig("gd_loss_curve.png", dpi=150)
    plt.show()
```

## 四、评估指标详解

### 4.1 三大指标

| 指标 | 公式 | 取值范围 | 含义 |
|------|------|----------|------|
| **MSE** | $\frac{1}{N}\sum(\hat{y}-y)^2$ | $[0, +\infty)$ | 均方误差，对大误差敏感 |
| **MAE** | $\frac{1}{N}\sum|\hat{y}-y|$ | $[0, +\infty)$ | 平均绝对误差，更鲁棒 |
| **R²** | $1 - \frac{SS_{res}}{SS_{tot}}$ | $(-\infty, 1]$ | 决定系数，越接近1越好 |

### 4.2 R² 的解释
- $R^2 = 1$：完美预测
- $R^2 = 0$：模型等同于"预测均值"
- $R^2 < 0$：模型比"预测均值"还差

---

## 五、解析解 vs 梯度下降对比

| 方法 | 优点 | 缺点 |
|------|------|------|
| **解析解** | 精确、一步到位 | 计算复杂度 $O(d^3)$，特征维度大时极慢 |
| **梯度下降** | 通用（可用于非线性模型）、适合大数据 | 需调学习率，收敛不保证全局最优 |

**实践结论：** 特征维度 $d < 10^4$ 时可用解析解；深度学习必用梯度下降（随机梯度下降 SGD）。

## 六、知识点总结

### 🌟 今日收获
- 线性回归虽简单，但它的**解析解、损失函数、梯度推导**是所有监督学习的基石
- 标准化（Normalization）对梯度下降至关重要：不标准化时不同特征量纲不同，梯度方向会偏向量纲大的特征
- `np.linalg.solve(A, b)` 比 `np.linalg.inv(A) @ b` 更数值稳定（推荐用前者）

### 核心公式
$$\boxed{\vec{\theta}^* = (X^TX + \lambda I)^{-1} X^T \vec{y}}$$
$$\boxed{\Delta\vec{w} = -\frac{\alpha}{N}X^T(\hat{y} - y) - \frac{\alpha\lambda}{N}\vec{w}}$$
