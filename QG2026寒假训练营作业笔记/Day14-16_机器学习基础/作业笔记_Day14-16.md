## 一、什么是机器学习？

### 1.1 传统编程 vs 机器学习

```
传统编程：规则 + 数据 → 结果
机器学习：数据 + 结果 → 规则（模型）
```

### 1.2 层级关系

$$\text{AI} \supset \text{机器学习 (ML)} \supset \text{深度学习 (DL)} \supset \text{大型语言模型 (LLM)}$$

### 1.3 机器学习三大类型

| 类型 | 特征 | 例子 |
|------|------|------|
| **有监督学习** | 有标注的 $(X, y)$ 对 | 房价预测（回归）、垃圾邮件识别（分类） |
| **无监督学习** | 只有 $X$，无标签 | 聚类、降维、异常检测 |
| **强化学习** | 智能体与环境交互，获得奖励信号 | 游戏AI、机器人控制 |

## 二、神经元与感知机

### 2.1 生物神经元 → 数学模型

```
输入信号 x₁, x₂, ..., xₙ
     ↓ 权重 w₁, w₂, ..., wₙ
   加权求和 z = w₁x₁ + w₂x₂ + ... + wₙxₙ + b
     ↓ 激活函数
   输出 a = f(z)
```

**数学表示：**
$$z = \vec{w}^T \vec{x} + b$$
$$a = f(z)$$

### 2.2 直接计算 vs 矩阵计算

```python
import numpy as np

# 单个神经元——直接计算
x = np.array([2.0, 3.0, 4.0])
w = np.array([0.5, -0.3, 0.8])
b = 0.1

z = sum(xi * wi for xi, wi in zip(x, w)) + b  # 或 np.dot(w, x) + b
print(f"加权和 z = {z:.4f}")

# 矩阵方式（更通用）
z_matrix = np.dot(w, x) + b
print(f"矩阵方式 z = {z_matrix:.4f}")  # 结果相同
```

## 三、激活函数

### 3.1 为什么需要激活函数？
没有激活函数，多层线性变换等价于一层线性变换 → 无法拟合非线性关系。

### 3.2 常见激活函数

#### Sigmoid
$$\sigma(z) = \frac{1}{1 + e^{-z}} \in (0, 1)$$

```python
def sigmoid(z):
    return 1 / (1 + np.exp(-z))

def sigmoid_deriv(z):
    s = sigmoid(z)
    return s * (1 - s)  # 导数：s(1-s)
```

**特点：** 输出在(0,1)，适合二分类输出层；**缺点：** 梯度消失（深层时导数趋近0）

#### ReLU（最常用）
$$\text{ReLU}(z) = \max(0, z)$$

```python
def relu(z):
    return np.maximum(0, z)

def relu_deriv(z):
    return (z > 0).astype(float)
```

**特点：** 计算简单，缓解梯度消失；**缺点：** "死亡 ReLU"问题（负值区域梯度为0）

#### Tanh
$$\tanh(z) = \frac{e^z - e^{-z}}{e^z + e^{-z}} \in (-1, 1)$$

```python
def tanh(z):
    return np.tanh(z)

def tanh_deriv(z):
    return 1 - np.tanh(z)**2
```

#### Softmax（多分类输出层）
$$\text{softmax}(z_i) = \frac{e^{z_i}}{\sum_j e^{z_j}}$$

```python
def softmax(z):
    # 数值稳定版（减去最大值防止溢出）
    e_z = np.exp(z - np.max(z))
    return e_z / e_z.sum()
```

## 四、前向传播

### 4.1 多层神经网络前向传播
$$Z^{[l]} = W^{[l]} A^{[l-1]} + b^{[l]}$$
$$A^{[l]} = f^{[l]}(Z^{[l]})$$

```python
def forward_pass(X, weights_list, biases_list, activations):
    """
    通用前向传播
    X: 输入，shape (n_features, batch_size)
    weights_list: 各层权重列表
    biases_list: 各层偏置列表
    activations: 各层激活函数列表
    """
    A = X
    caches = []
    
    for W, b, f in zip(weights_list, biases_list, activations):
        Z = W @ A + b
        A = f(Z)
        caches.append((A, Z, W, b))
    
    return A, caches
```

### 4.2 手写实现：3层神经网络（分类任务）
```python
"""Day 14-16 练习：神经网络前向传播实现  日期：2026年2月27日-3月1日"""
import numpy as np

# 激活函数
def sigmoid(z): return 1 / (1 + np.exp(-z))
def relu(z): return np.maximum(0, z)
def softmax(z):
    e_z = np.exp(z - np.max(z, axis=0, keepdims=True))
    return e_z / e_z.sum(axis=0, keepdims=True)

def sigmoid_deriv(z):
    s = sigmoid(z)
    return s * (1 - s)

def relu_deriv(z):
    return (z > 0).astype(float)

class NeuralNetwork:
    """3层全连接网络：输入层(n) → 隐藏层1(h1) → 隐藏层2(h2) → 输出层(out)"""
    
    def __init__(self, layer_sizes: list, seed: int = 42):
        """
        layer_sizes: 各层神经元数，例如 [4, 8, 4, 3] 表示
                     4个输入 → 8个隐藏1 → 4个隐藏2 → 3个输出
        """
        np.random.seed(seed)
        self.layer_sizes = layer_sizes
        self.weights = []
        self.biases = []
        
        # He 初始化（ReLU 层推荐）
        for i in range(len(layer_sizes) - 1):
            n_in = layer_sizes[i]
            n_out = layer_sizes[i + 1]
            W = np.random.randn(n_out, n_in) * np.sqrt(2 / n_in)
            b = np.zeros((n_out, 1))
            self.weights.append(W)
            self.biases.append(b)
    
    def forward(self, X):
        """
        X: shape (n_features, batch_size)
        返回最终输出和中间缓存
        """
        A = X
        self.caches = []
        
        # 前 N-1 层用 ReLU
        for W, b in zip(self.weights[:-1], self.biases[:-1]):
            Z = W @ A + b
            A = relu(Z)
            self.caches.append((A, Z))
        
        # 最后一层用 Softmax（多分类）
        Z_out = self.weights[-1] @ A + self.biases[-1]
        A_out = softmax(Z_out)
        self.caches.append((A_out, Z_out))
        
        return A_out
    
    def predict(self, X):
        """返回预测类别"""
        probs = self.forward(X)
        return np.argmax(probs, axis=0)


# ── 测试 ──
if __name__ == "__main__":
    # 网络结构：4输入 → 8隐藏 → 3输出（3分类）
    nn = NeuralNetwork([4, 8, 3])
    
    # 随机输入（batch_size=5）
    X = np.random.randn(4, 5)
    output = nn.forward(X)
    
    print("网络结构：", nn.layer_sizes)
    print(f"输入形状：{X.shape}")
    print(f"输出形状：{output.shape}")
    print("输出（概率）：\n", output.round(4))
    print("每列概率之和：", output.sum(axis=0).round(6))  # 应接近 1.0
    print("预测类别：", nn.predict(X))
```

## 五、反向传播（Backpropagation）

### 5.1 核心思想
利用**链式法则**，从输出层向输入层逐层传播误差梯度。

### 5.2 损失函数

**均方误差（MSE）**——回归任务：
$$\mathcal{L} = \frac{1}{N}\sum_{i=1}^{N}(\hat{y}_i - y_i)^2$$

**交叉熵损失（CE）**——分类任务：
$$\mathcal{L} = -\frac{1}{N}\sum_{i=1}^{N}\sum_{k=1}^{K} y_{ik}\log\hat{y}_{ik}$$

```python
def cross_entropy_loss(y_pred, y_true):
    """
    y_pred: (K, N)，每列是一个样本的各类概率
    y_true: (N,)，每个样本的真实类别标签
    """
    N = y_true.shape[0]
    # 取每个样本真实类别的预测概率
    correct_probs = y_pred[y_true, np.arange(N)]
    loss = -np.mean(np.log(correct_probs + 1e-8))
    return loss
```

### 5.3 反向传播推导（简化版）

对于最后一层（Softmax + CrossEntropy），梯度简化为：
$$\frac{\partial \mathcal{L}}{\partial Z^{[L]}} = A^{[L]} - Y$$

```python
def backward_single_layer(dA, cache, activation="relu"):
    """单层反向传播"""
    A_prev, Z, W, b = cache
    m = A_prev.shape[1]  # batch size
    
    if activation == "relu":
        dZ = dA * relu_deriv(Z)
    elif activation == "sigmoid":
        dZ = dA * sigmoid_deriv(Z)
    
    dW = (dZ @ A_prev.T) / m
    db = dZ.mean(axis=1, keepdims=True)
    dA_prev = W.T @ dZ
    
    return dA_prev, dW, db
```

## 六、正则化与过拟合

### 6.1 过拟合 vs 欠拟合

| 问题 | 训练误差 | 测试误差 | 原因 |
|------|----------|----------|------|
| 欠拟合 | 高 | 高 | 模型太简单 |
| 合适 | 低 | 低 | ✅ |
| 过拟合 | 极低 | 高 | 模型太复杂，记住了噪声 |

### 6.2 正则化方法

**L2 正则化（权重衰减）：**
$$\mathcal{L}_{total} = \mathcal{L} + \frac{\lambda}{2N}\sum_{l}\|W^{[l]}\|_F^2$$

```python
# 在损失函数中加 L2 惩罚
lambda_reg = 0.01
l2_penalty = sum(np.sum(W**2) for W in nn.weights)
total_loss = cross_entropy + (lambda_reg / 2) * l2_penalty
```

**Dropout：** 训练时随机让某些神经元"失活"（概率为 p）
```python
def dropout(A, keep_prob=0.8):
    """训练时使用 Dropout"""
    mask = np.random.binomial(1, keep_prob, size=A.shape) / keep_prob
    return A * mask  # 乘以 mask 并缩放（反向 Dropout）
```

**批归一化（Batch Normalization）：** 每层输出归一化，加速收敛

## 七、知识点总结

### 神经网络关键概念速查

| 概念 | 公式/代码 | 作用 |
|------|-----------|------|
| 线性变换 | $Z = WX + b$ | 特征提取 |
| 激活函数 | $A = f(Z)$ | 引入非线性 |
| 前向传播 | 从输入到输出 | 推理/预测 |
| 损失函数 | MSE, CrossEntropy | 衡量预测误差 |
| 反向传播 | 链式法则 | 计算梯度 |
| 梯度下降 | $W \leftarrow W - \alpha \nabla_W \mathcal{L}$ | 更新参数 |
| 正则化 | L2, Dropout | 防止过拟合 |

- 理解了为什么 ReLU 比 Sigmoid 更常用（梯度消失问题）

### 💡 AI 学习关键公式
