## 一、什么是聚类？

### 1.1 聚类 vs 分类

| | 聚类（Clustering） | 分类（Classification） |
|--|--------------------|-----------------------|
| 标签 | **无**标注（无监督） | **有**标注（有监督） |
| 目标 | 发现数据中的自然分组 | 将数据分配到已知类别 |
| 应用 | 用户画像、基因分析 | 垃圾邮件、图像识别 |

### 1.2 K-means 的目标
最小化**惯量（Inertia）**——所有点到所在簇中心的距离平方和：

$$J = \sum_{k=1}^{K} \sum_{\vec{x} \in C_k} \|\vec{x} - \vec{\mu}_k\|^2$$

---

## 二、K-means 算法

### 2.1 算法步骤
```
1. 随机初始化 K 个簇中心 μ₁, μ₂, ..., μK
2. 重复直到收敛：
   a. 分配步骤：将每个点分配到最近的中心
      C_k = {x : ‖x - μ_k‖² ≤ ‖x - μ_j‖² ∀j}
   b. 更新步骤：重新计算每个簇的中心
      μ_k = (1/|C_k|) Σ x
3. 终止条件：中心不再移动 或 达到最大迭代次数
```

### 2.2 K-means++ 初始化
原始随机初始化容易陷入局部最优。K-means++ 改进：

```
1. 随机选第一个中心
2. 对每个点，计算其到最近已有中心的距离 D(x)
3. 以概率正比于 D(x)² 选下一个中心
4. 重复直到选出 K 个中心
```

**效果：** 初始中心更分散，收敛更快，结果更好。

## 三、从零实现 K-means

```python
"""Day 19-20 练习：K-means 聚类算法（从零实现）日期：2026年3月4日-5日
要求：仅用 numpy，实现 fit/predict/fit_predict，实现肘部法则
数据集：鸢尾花（Iris）"""

import numpy as np
import matplotlib.pyplot as plt
import matplotlib
matplotlib.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei']
matplotlib.rcParams['axes.unicode_minus'] = False


class KMeans:
    """K-means 聚类算法（numpy 实现）"""
    
    def __init__(self, n_clusters: int = 3, max_iter: int = 300,
                 tol: float = 1e-4, init: str = "k-means++",
                 n_init: int = 10, random_state: int = 42):
        """
        n_clusters: 簇数 K
        max_iter: 最大迭代次数
        tol: 收敛阈值（中心移动量小于此值时停止）
        init: 初始化方式，"random" 或 "k-means++"
        n_init: 多次随机初始化，取最优结果
        random_state: 随机种子
        """
        self.n_clusters = n_clusters
        self.max_iter = max_iter
        self.tol = tol
        self.init = init
        self.n_init = n_init
        self.random_state = random_state
        self.cluster_centers_ = None
        self.labels_ = None
        self.inertia_ = None
    
    def _init_centers(self, X: np.ndarray, rng) -> np.ndarray:
        """初始化簇中心"""
        N = X.shape[0]
        
        if self.init == "random":
            idx = rng.choice(N, self.n_clusters, replace=False)
            return X[idx].copy()
        
        elif self.init == "k-means++":
            # K-means++ 初始化
            centers = [X[rng.integers(0, N)]]
            
            for _ in range(1, self.n_clusters):
                # 计算每个点到最近中心的距离
                dists = np.array([
                    min(np.sum((x - c)**2) for c in centers)
                    for x in X
                ])
                # 按距离平方的概率采样
                probs = dists / dists.sum()
                cum_probs = np.cumsum(probs)
                r = rng.random()
                idx = np.searchsorted(cum_probs, r)
                centers.append(X[idx])
            
            return np.array(centers)
        
        else:
            raise ValueError(f"未知初始化方式：{self.init}")
    
    def _assign_labels(self, X: np.ndarray, centers: np.ndarray) -> np.ndarray:
        """分配步骤：将每个点分配到最近的中心"""
        # 利用广播计算所有点到所有中心的距离
        # X: (N, d), centers: (K, d) → dists: (N, K)
        dists = np.sum((X[:, np.newaxis, :] - centers[np.newaxis, :, :])**2, axis=2)
        return np.argmin(dists, axis=1)
    
    def _update_centers(self, X: np.ndarray, labels: np.ndarray) -> np.ndarray:
        """更新步骤：重新计算每个簇的中心"""
        new_centers = np.zeros((self.n_clusters, X.shape[1]))
        for k in range(self.n_clusters):
            mask = labels == k
            if mask.sum() > 0:
                new_centers[k] = X[mask].mean(axis=0)
            else:
                # 空簇处理：随机重新分配
                new_centers[k] = X[np.random.randint(X.shape[0])]
        return new_centers
    
    def _compute_inertia(self, X: np.ndarray, labels: np.ndarray,
                          centers: np.ndarray) -> float:
        """计算惯量"""
        inertia = 0.0
        for k in range(self.n_clusters):
            mask = labels == k
            if mask.sum() > 0:
                inertia += np.sum((X[mask] - centers[k])**2)
        return inertia
    
    def _fit_single(self, X: np.ndarray, rng):
        """单次 K-means 迭代"""
        centers = self._init_centers(X, rng)
        labels = None
        
        for _ in range(self.max_iter):
            # 分配
            new_labels = self._assign_labels(X, centers)
            # 更新
            new_centers = self._update_centers(X, new_labels)
            # 检查收敛
            if np.linalg.norm(new_centers - centers) < self.tol:
                centers = new_centers
                labels = new_labels
                break
            centers = new_centers
            labels = new_labels
        
        inertia = self._compute_inertia(X, labels, centers)
        return centers, labels, inertia
    
    def fit(self, X: np.ndarray):
        """多次初始化，取最优"""
        rng = np.random.default_rng(self.random_state)
        
        best_inertia = np.inf
        best_centers = None
        best_labels = None
        
        for _ in range(self.n_init):
            centers, labels, inertia = self._fit_single(X, rng)
            if inertia < best_inertia:
                best_inertia = inertia
                best_centers = centers
                best_labels = labels
        
        self.cluster_centers_ = best_centers
        self.labels_ = best_labels
        self.inertia_ = best_inertia
        return self
    
    def predict(self, X: np.ndarray) -> np.ndarray:
        """预测新样本的簇标签"""
        return self._assign_labels(X, self.cluster_centers_)
    
    def fit_predict(self, X: np.ndarray) -> np.ndarray:
        """训练并返回训练集的标签"""
        self.fit(X)
        return self.labels_


# ── 肘部法则（Elbow Method）──
def elbow_method(X: np.ndarray, k_range=range(1, 11)):
    """绘制肘部曲线，帮助选择最优 K"""
    inertias = []
    for k in k_range:
        km = KMeans(n_clusters=k, n_init=5, random_state=42)
        km.fit(X)
        inertias.append(km.inertia_)
    
    plt.figure(figsize=(8, 5))
    plt.plot(list(k_range), inertias, "bo-", markersize=8, linewidth=2)
    plt.title("肘部法则：选择最优 K 值", fontsize=14, fontweight="bold")
    plt.xlabel("簇数 K")
    plt.ylabel("惯量（Inertia）")
    plt.grid(True, alpha=0.3)
    plt.xticks(list(k_range))
    plt.tight_layout()
    plt.savefig("elbow_curve.png", dpi=150)
    plt.show()
    
    return inertias


# ── 实验：鸢尾花数据集 ──

    # ── 肘部法则 ──
    print("=== 肘部法则 ===")
```

## 四、聚类评估指标

### 4.1 无标签评估

**轮廓系数（Silhouette Score）** — 范围 [-1, 1]，越接近1越好：
$$s(i) = \frac{b(i) - a(i)}{\max(a(i), b(i))}$$

- $a(i)$：点 $i$ 到同簇其他点的平均距离（紧密度）
- $b(i)$：点 $i$ 到最近其他簇中所有点的平均距离（分离度）

```python
from sklearn.metrics import silhouette_score, davies_bouldin_score

sil = silhouette_score(X_scaled, labels)
print(f"轮廓系数：{sil:.4f}")  # 越大越好

db = davies_bouldin_score(X_scaled, labels)
print(f"Davies-Bouldin 指数：{db:.4f}")  # 越小越好
```

### 4.2 有标签评估（用于验证时）
```python
from sklearn.metrics import adjusted_rand_score, normalized_mutual_info_score

ari = adjusted_rand_score(y_true, labels)    # 调整兰德指数
nmi = normalized_mutual_info_score(y_true, labels)  # 标准化互信息
print(f"ARI：{ari:.4f}")  # 1 = 完美，0 = 随机
print(f"NMI：{nmi:.4f}")
```

## 五、知识点总结

### K-means 的局限性

| 局限 | 说明 | 解决方案 |
|------|------|----------|
| 需预先指定 K | 不知道有几个簇 | 肘部法则、轮廓系数 |
| 对初值敏感 | 不同初始值结果不同 | K-means++ + n_init 多次运行 |
| 只能识别球形簇 | 非凸形状效果差 | DBSCAN、高斯混合模型 |
| 对异常值敏感 | 离群点拉偏中心 | K-medoids |

### 🌟 今日收获
- K-means 的核心思想是**交替优化**：固定中心求最优分配，固定分配求最优中心
- K-means++ 的概率采样策略让初始化更聪明，是工程中的默认选择
- `n_init=10` 多次运行取最优是防止局部最优的实用技巧

### 💡 K-means 在 AI 中的应用
- 图像颜色量化（将图像颜色压缩到 K 种）
- 词向量聚类（NLP）
- 推荐系统中的用户分群
