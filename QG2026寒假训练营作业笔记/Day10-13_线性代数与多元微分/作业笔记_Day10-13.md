## 第一部分：线性代数

### 一、向量基础

#### 1.1 向量的几何意义
- **向量** = 有方向、有大小的量
- $\vec{a} = (a_1, a_2, ..., a_n)^T$，表示 $n$ 维空间中的一个点/方向

#### 1.2 向量运算
$$\vec{a} \cdot \vec{b} = a_1b_1 + a_2b_2 + \cdots + a_nb_n = \|\vec{a}\|\|\vec{b}\|\cos\theta$$

**点积的几何意义：** $\vec{a} \cdot \vec{b} = 0$ 表示两向量垂直（正交）。

```python
import numpy as np

a = np.array([3.0, 4.0])
b = np.array([1.0, 2.0])

# 点积
dot_product = np.dot(a, b)  # 3*1 + 4*2 = 11

# 模长
norm_a = np.linalg.norm(a)  # sqrt(9+16) = 5

# 夹角
cos_theta = np.dot(a, b) / (np.linalg.norm(a) * np.linalg.norm(b))
angle = np.degrees(np.arccos(cos_theta))

# 单位向量
unit_a = a / np.linalg.norm(a)
```

#### 1.3 线性组合与线性相关
- **线性组合**：$c_1\vec{v_1} + c_2\vec{v_2} + \cdots + c_n\vec{v_n}$
- **张成空间（Span）**：所有线性组合构成的空间
- **线性无关**：没有一个向量可以由其他向量的线性组合表示

```python
# 判断线性相关：秩 < 向量数
vectors = np.column_stack([np.array([1,2,3]), np.array([4,5,6]), np.array([7,8,9])])
rank = np.linalg.matrix_rank(vectors)
print(f"秩：{rank}，向量数：3 → {'线性相关' if rank < 3 else '线性无关'}")
```

### 二、矩阵运算

#### 2.1 矩阵乘法
$$[AB]_{ij} = \sum_k A_{ik} B_{kj}$$

- **条件**：$A$ 的列数 = $B$ 的行数
- **形状**：$(m \times k) \times (k \times n) = (m \times n)$
- **非交换**：$AB \neq BA$（一般情况）

```python
A = np.array([[1, 2], [3, 4]])
B = np.array([[5, 6], [7, 8]])

C = A @ B    # 矩阵乘法
# C = [[19, 22],
#      [43, 50]]
```

#### 2.2 矩阵转置
$$(AB)^T = B^T A^T$$
$$(A^T)^T = A$$

```python
A_T = A.T
print("A =\n", A)
print("A^T =\n", A_T)
# 对称矩阵：A == A.T
```

#### 2.3 逆矩阵
若 $AB = BA = I$，则 $B = A^{-1}$

**存在条件：** $\det(A) \neq 0$（满秩方阵）

$$(AB)^{-1} = B^{-1}A^{-1}$$

```python
A = np.array([[1.0, 2.0], [3.0, 4.0]])
det_A = np.linalg.det(A)           # -2.0
A_inv = np.linalg.inv(A)

# 验证
I = A @ A_inv
print(np.allclose(I, np.eye(2)))   # True

# 解方程组 Ax = b（比求逆更稳定）
b = np.array([1.0, 2.0])
x = np.linalg.solve(A, b)
```

#### 2.4 行列式
- $2 \times 2$：$\det\begin{pmatrix}a & b \\ c & d\end{pmatrix} = ad - bc$
- 几何意义：行列式的绝对值 = 矩阵所有行向量构成的平行多面体的体积
- $\det(A) = 0$ → 矩阵奇异（不可逆）

```python
# 行列式
det = np.linalg.det(A)

# 迹（对角元素之和 = 特征值之和）
trace = np.trace(A)   # 1 + 4 = 5
```

#### 2.5 特征值与特征向量
$$A\vec{v} = \lambda\vec{v}$$

- $\lambda$：特征值（缩放因子）
- $\vec{v}$：特征向量（方向不变的向量）

```python
eigenvalues, eigenvectors = np.linalg.eig(A)
print("特征值：", eigenvalues)
print("特征向量（列向量）：\n", eigenvectors)

# 验证：A @ v = λ * v
for i, (lam, v) in enumerate(zip(eigenvalues, eigenvectors.T)):
    print(f"λ{i} = {lam:.4f}, 验证: {np.allclose(A @ v, lam * v)}")
```

#### 2.6 SVD 奇异值分解
$$A = U \Sigma V^T$$

- $U$：左奇异向量（正交矩阵）
- $\Sigma$：奇异值对角矩阵
- $V^T$：右奇异向量

**应用：** 图像压缩、主成分分析（PCA）、推荐系统

```python
U, s, Vt = np.linalg.svd(A)
# 重建矩阵
A_reconstruct = U @ np.diag(s) @ Vt
print(np.allclose(A, A_reconstruct))  # True

# 低秩近似（图像压缩原理）：只用前 k 个奇异值
k = 1
A_approx = U[:, :k] @ np.diag(s[:k]) @ Vt[:k, :]
```

## 第二部分：多元微分

### 三、偏导数与梯度

#### 3.1 偏导数
对多元函数 $f(x, y)$，偏导数描述某一方向的变化率：

$$\frac{\partial f}{\partial x} = \lim_{\Delta x \to 0} \frac{f(x+\Delta x, y) - f(x, y)}{\Delta x}$$

```python
# 数值偏导（有限差分法）
def partial_derivative(f, var_idx, point, h=1e-5):
    """计算函数 f 在 point 处对第 var_idx 个变量的偏导数"""
    p1 = list(point)
    p2 = list(point)
    p1[var_idx] += h
    p2[var_idx] -= h
    return (f(*p1) - f(*p2)) / (2 * h)

# 示例：f(x,y) = x² + 2xy + y²
f = lambda x, y: x**2 + 2*x*y + y**2
# ∂f/∂x = 2x + 2y，在 (1, 2) 处 = 6
df_dx = partial_derivative(f, 0, (1.0, 2.0))
df_dy = partial_derivative(f, 1, (1.0, 2.0))
```

#### 3.2 梯度
函数在某点处**上升最快**的方向：

$$\nabla f(x, y) = \begin{pmatrix} \frac{\partial f}{\partial x} \\ \frac{\partial f}{\partial y} \end{pmatrix}$$

```python
# 解析梯度
def gradient_f(x, y):
    """f(x,y) = x² + y² 的梯度"""
    return np.array([2*x, 2*y])

# 梯度方向可视化
import matplotlib.pyplot as plt
x = np.linspace(-3, 3, 20)
y = np.linspace(-3, 3, 20)
X, Y = np.meshgrid(x, y)
Z = X**2 + Y**2  # 碗形函数

fig, axes = plt.subplots(1, 2, figsize=(12, 5))
# 等高线图
cs = axes[0].contour(X, Y, Z, levels=15, cmap="viridis")
axes[0].quiver(X, Y, 2*X, 2*Y, alpha=0.5, color="red")
axes[0].set_title("梯度场（红箭头）与等高线")
axes[0].clabel(cs)

# 3D 曲面
from mpl_toolkits.mplot3d import Axes3D
ax3d = fig.add_subplot(111, projection="3d")
ax3d.plot_surface(X, Y, Z, cmap="viridis", alpha=0.8)
ax3d.set_title("f(x,y) = x² + y²")
```

### 四、链式法则与梯度下降

#### 4.1 链式法则
$$\frac{dz}{dx} = \frac{dz}{dy} \cdot \frac{dy}{dx}$$

多元版本：
$$\frac{\partial z}{\partial x} = \sum_i \frac{\partial z}{\partial y_i} \cdot \frac{\partial y_i}{\partial x}$$

**重要性：** 神经网络反向传播的数学基础！

#### 4.2 梯度下降的直觉
沿梯度的**反方向**走一小步，函数值下降最快：

$$x_{t+1} = x_t - \alpha \nabla f(x_t)$$

- $\alpha$：学习率（步长）
- 太大 → 震荡不收敛；太小 → 收敛慢

```python
def gradient_descent_demo():
    """梯度下降求 f(x,y) = x² + y² 的最小值"""
    # 解析解显然是 (0, 0)，这里用梯度下降验证
    x, y = 3.0, 4.0  # 初始点
    alpha = 0.1       # 学习率
    
    history = [(x, y)]
    for _ in range(50):
        grad_x = 2 * x
        grad_y = 2 * y
        x -= alpha * grad_x
        y -= alpha * grad_y
        history.append((x, y))
    
    print(f"最终点：({x:.6f}, {y:.6f})")  # 接近 (0, 0)
    return history

history = gradient_descent_demo()
```

```python
"""Day 10-13 实验：线性代数 + 多元微分 NumPy 实验  日期：2026年2月23-26日"""
import numpy as np
import matplotlib.pyplot as plt

# ── 向量实验 ──
a = np.array([3.0, 4.0])
b = np.array([1.0, 2.0])
print(f"a·b = {np.dot(a, b)}")
print(f"夹角 = {np.degrees(np.arccos(np.dot(a,b)/(np.linalg.norm(a)*np.linalg.norm(b)))):.2f}°")

# ── 矩阵运算实验 ──
A = np.array([[1,2],[3,4]], dtype=float)
B = np.array([[5,6],[7,8]], dtype=float)
print("AB =\n", A @ B)
print("det(A) =", np.linalg.det(A))
print("inv(A) =\n", np.linalg.inv(A))

# ── 特征分解 ──
eigvals, eigvecs = np.linalg.eig(A)
print("特征值：", eigvals)

# ── 解方程组 ──
# 系统：x + 2y = 5, 3x + 4y = 6
coeffs = np.array([[1,2],[3,4]])
rhs = np.array([5.0, 6.0])
solution = np.linalg.solve(coeffs, rhs)
print(f"解：x={solution[0]:.2f}, y={solution[1]:.2f}")  # x=-4, y=4.5

# ── 梯度下降可视化 ──
x_vals = [3.0]
y_vals = [4.0]
x, y = 3.0, 4.0
alpha = 0.1
for _ in range(30):
    x -= 0.1 * 2 * x
    y -= 0.1 * 2 * y
    x_vals.append(x)
    y_vals.append(y)

fig, ax = plt.subplots(figsize=(8, 6))
X, Y = np.meshgrid(np.linspace(-4, 4, 100), np.linspace(-5, 5, 100))
Z = X**2 + Y**2
ax.contour(X, Y, Z, levels=20, cmap="Blues")
ax.plot(x_vals, y_vals, "r.-", markersize=8, label="梯度下降路径")
ax.plot(x_vals[0], y_vals[0], "go", markersize=12, label="起点")
ax.plot(x_vals[-1], y_vals[-1], "r*", markersize=12, label="终点")
ax.set_title("梯度下降轨迹（f(x,y) = x²+y²）")
ax.legend()
plt.savefig("gradient_descent.png", dpi=150)
plt.show()
```
